GL Mapping — Process Steps
Complete these steps, then return to submit your ticket.
⚠️ Before submitting, the GL mapping must be verified and budgets must tie. Complete the steps below then start the process over.
  • 1Log into RealPage (or the relevant PMS) and navigate to the Budget module.
  • 2Open the GL Mapping screen for the applicable property.
  • 3Review each GL code in the budget against the client's chart of accounts.
  • 4For any unmapped GLs, assign the correct client GL code from the dropdown.
  • 5Save all mappings and run the budget reconciliation report.
  • 6Verify the totals tie between the RPM budget and the client GL structure.
  • 7Once complete and totals tie, close this window and click Submit Approval to restart.
